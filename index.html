BACKEND

EXPRESS

  • Express is a minimal and flexible Node.js web application framework.
  • Setting up Express: const express = require('express'); const app = express();
  • Routing: Define routes using app.get, app.post, app.put, and app.delete.
  • Using Middleware: Apply middleware with app.use.
  • Creating Custom Middleware: Create functions that have access to req, res, and next.
  • Handling Errors in Express: Use middleware for error handling with four parameters (err, req, res, next).
  •  Keep your code modular, use environment variables for sensitive data, and write clean, readable code.

MONGO DB

  • What is MongoDB? MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. Installing MongoDB: Follow the instructions on the official MongoDB website. Commands: Use MongoDB shell or drivers to interact with the database.

FRONTEND

REACT

  • React is a JavaScript library for building user interfaces, focusing on the component-based architecture.
  • Setting up a React Project: Use tools like Create React App to quickly set up a new React project with all the necessary dependencies.
  • Components: React applications are built using components, which are reusable, self-contained modules. There are two types of components: Functional and Class components.
  • Props and State: Components can receive data through props and manage their internal state using the useState hook or class-based state.
  • Hooks: React provides several built-in hooks to manage component lifecycle and state. Commonly used hooks include useState, useEffect, useContext, useReducer, useRef, useMemo, and useCallback.
  • Routing: For single-page applications (SPAs), React Router is a popular library for handling client-side routing. It allows you to define different routes and navigate between them seamlessly. React Router also supports nested routes for complex applications.

BOOTSTRAP

  • Popular CSS framework for building responsive, mobile-first websites.
  • Installation: Add via CDN or npm: npm install bootstrap
  • Grid System: 12-column grid for responsive layouts.
  • Utility Classes: Extensive set of utilities for styling and layout.
  • Components: Pre-built UI components like buttons, forms, modals.

TAILWIND

  • Utility-first CSS framework for creating custom designs without writing CSS.
  • Installation: Add via CDN or npm: npm install tailwindcss
  • Utility Classes: Extensive set of utilities for styling.
  • Responsive Design: Built-in responsive utilities.
  • Customization: Highly configurable via tailwind.config.js.
